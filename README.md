# 🤖 AI ChatBot using LangChain, Hugging Face & Python

> An intelligent AI-powered chatbot built with **Python**, **LangChain**, and **Hugging Face**. This project demonstrates how to integrate Large Language Models (LLMs) into a Python application to create natural, context-aware conversations using prompt engineering and modern AI frameworks.

---

## 📖 Overview

This project is a simple yet powerful AI chatbot that leverages the capabilities of Large Language Models (LLMs) through the LangChain framework and Hugging Face Inference API.

The chatbot accepts user queries, sends them to a pre-trained language model, and generates intelligent, human-like responses. The project serves as a great starting point for anyone learning Generative AI, Prompt Engineering, and LangChain.

Whether you're a beginner exploring LLMs or a developer building conversational AI applications, this project demonstrates the complete workflow of integrating an AI model into a Python application.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/devabhinavraj/AI-ChatBot.git
```

Move into the project directory

```bash
cd langchain-ai-chatbot
```

---

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

or

```bash
pip install langchain
pip install langchain-huggingface
pip install python-dotenv
pip install huggingface_hub
```

---

## 🔑 Environment Variables

Create a **.env** file in the project directory.

```env
HUGGINGFACEHUB_API_TOKEN=your_api_key_here
```

## 🧠 How It Works

1. User enters a prompt.
2. LangChain sends the prompt to Hugging Face.
3. The Large Language Model processes the request.
4. The chatbot generates a human-like response.
5. The response is displayed to the user.
